# Project Overview

## Vision
Pilotarr - A unified application to manage *arr stack services (Radarr, Sonarr, qBittorrent) and media platforms (Jellyfin, Jellyseerr). Provides a single dashboard for home media server management, monitoring, and analytics.

## Goals
- [ ] Unified dashboard for all *arr services
- [ ] Media library management and statistics (Jellyfin)
- [ ] Request management (Jellyseerr)
- [ ] Torrent monitoring and management (qBittorrent)
- [ ] Calendar view for upcoming media
- [ ] Alerts and notifications
- [ ] System monitoring and analytics

## Non-Goals
- Not a replacement for individual *arr apps (it's a unified layer)
- Not a media player or streaming service

## Success Metrics
- All connected services reachable and data displayed in real-time
- Fast page loads and responsive UI
- Reliable background sync via schedulers
